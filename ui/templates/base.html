Yelpminer

Yelpminer helps you build REST API request for your Yelp data.
1. Check any item you want to query.
2. Fill input values to query with.
3. Click the "Build Request" button at the bottom to get HTTP request.
4. You can also view data directly on browser by clicking "View on Browser" button.
